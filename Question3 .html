Define Call with example

Explaination

call is basically a method with the help of these we invoke the function,and it takes two paramters first parameter will be ( the context parameter ,
anything passed here is accessible using this inside the function body)
and second parameter takes (the value to the function itself) 

        var obj = {num:2};
        var addToThis = function(a){

          return this.num + a;
        }
        console.log(addToThis.call(obj,3));

Define Bind with example

Explaination

call is basically a method with the help of these we invoke the function,and it takes two paramters first parameter will be ( the context parameter ,
anything passed here is accessible using this inside the function body)
and second parameter takes (the value to the function itself) 

          var person =  {
            name:'sunny',
            print:function(){
              console.log('Name is :',this.name);
            }
          };


          var abc = person.print;
          abc = abc.bind(person);
          console.log(abc());
